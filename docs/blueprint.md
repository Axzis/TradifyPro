# **App Name**: Tradify Pro

## Core Features:

- Project Setup: Set up the Next.js project with necessary dependencies and Firebase integration.
- Firebase Authentication: Implement Firebase authentication with Email/Password provider.
- Data Structure Definition: Define TypeScript types for Trade and EquityTransaction data.
- Firestore Security Rules: Implement Firestore security rules to enforce data ownership.
- Public & Auth Pages: Implement public pages (landing, login, register) and authentication state management.
- Theme Management: Implement theme management using next-themes.
- Dashboard Layout: Set up the dashboard layout with sidebar navigation and placeholder pages.
- Tag Input Component: Create a basic Tag Input UI component.
- Dashboard UI: Implement the static UI for the dashboard page (privacy, filters).
- Trade History UI: Implement the static UI for the trade history page (filters, detail dialog).
- New Trade Form UI: Update the NewTradeForm with new fields (R-Multiple, ImageKit) and TagInput components.
